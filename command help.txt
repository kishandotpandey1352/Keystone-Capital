Day-1 run commands (exact)

Note: 1>Run the docker windows app

2> From repo root:

copy .env.example .env
docker-compose up --build #DOCKER COMMAND TO CREATE DOCKER IMAGES FROM DOCKER FILE


3> In a second terminal:

docker-compose exec api alembic upgrade head #Bring the database schema to the latest version.


4> Frontend:

cd frontend
npm install
npm start


Validation:

API: http://localhost:8000/health returns {"status":"ok"}

WebSocket: frontend shows connected and echo works
